# SimGEKI 在线设置工具

!> 当前在线设置工具仅支持 SimGETRO/SimGEKI 系列产品，不适用于 SimPad 系列产品。

SimGEKI WebControl 是适用于 SimGETRO/SimGEKI 系列产品的基于 Web 的在线设备固件管理与配置工具。为相关用户提供了基本的工作模式切换、摇杆校准、输入和读卡器测试以及固件管理功能。

- [GitHub仓库地址](https://github.com/SimDevices-Project/SimGEKI-WebControl)

> 工具大部分使用 AI 工具进行开发，可能存在大量 BUG 或问题，欢迎前往 GitHub 提出 Issue。

## 设备配置

在设备配置标签页，您可以对您的设备进行工作模式切换、序列号读取以及进行简单的摇杆和灯光测试。

### 设备连接

要使用设备配置页的任何功能，您必须使用“连接设备”按钮，并允许您的浏览器获取您的USB设备列表。正确的设备名称应当是“SimGEKI Config”，请注意甄别。

### 摇杆配置

**校准摇杆**：每台设备出厂前均经过了校准，但如果您重新安装过摇杆、或是更新固件，则必须重新校准摇杆。要校准摇杆，您需要将您的摇杆居中摆放，然后点击“校准摇杆”按钮。最后，别忘记保存。

**读取摇杆信息**：获取原始摇杆输入数据和处理后数据，方便您分析摇杆状态。读取后的信息会显示在“设备状态”部分。

### 输入模式

**读取当前输入模式**：从您的设备读取当前工作模式并在下方下拉框显示，目前可能值为“IO4兼容”、“DLL输入”、“模拟键盘”。对应了不同的设备工作模式。

**设置输入模式**：让设备工作在指定的工作模式下，请注意保存后才会不会在断电后丢失您的配置。

### 调试

**开启实时监控**：快速持续从设备获取摇杆原始输入数据并展示。

**灯光测试**：发送灯光测试指令，让您的设备上所有灯组按照“红、绿、蓝、白”四色依次显示灯光。

### 保存或放弃

**保存配置**：您只有在点击保存配置按钮后，才不会在设备断电后丢失本次的配置。

**放弃配置**：丢弃所有修改的配置，重新加载上次保存的配置。

### 设备状态

该窗口会在您成功连接到设备后显示连接状态、序列号和摇杆状态。

### 配置日志

该窗口会展示您在本页面操作中留下的日志。

## IO4测试

IO4测试标签页为任何IO4兼容设备提供了标准的输入测试功能，您可以直接在网页上测试您的设备按键输入情况。

## 读卡器测试

读卡器测试标签页为任何兼容读卡器提供了基本的测试功能，其在连接到对应串口后可以模拟兼容上位机通信来测试基本的兼容读卡器功能。
