## 组装设备

在正式使用前，您需要遵循以下步骤将侧键安装至您的主机上

1. 拆除任意一侧的侧键包装袋，取出侧键
2. 使用随附的H2.0内六角扳手移除侧键保护板的5颗螺丝，拆下保护板
3. 根据线缆标识，按组连接对应的侧键线缆
4. 将侧键对齐主机，对准螺丝孔，请注意不要让线材被铝型材和侧键夹住
5. 使用随附的螺丝和扳手将侧键固定在主机上
6. 将侧键保护板重新安装在侧键上
7. 将另一侧侧键使用相同方式安装完毕
