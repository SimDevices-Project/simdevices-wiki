## 组装设备

在正式使用前，您需要遵循以下步骤将侧键安装至您的主机上。

1. 拆除任意一侧的侧键包装袋，取出侧键
2. 使用随附的H2.5内六角扳手移除侧键装饰板的两颗平头螺丝，拆下装饰板
3. 连接侧键线缆
4. 将侧键对齐主机，对准螺丝孔，请注意不要让线材被框体或侧键夹住
5. 使用圆头内六角螺丝对准底部两行中间的两个螺丝孔，使用H2.5内六角扳手拧紧
7. 将侧键装饰板的保护膜移除，对齐侧键，使用平头螺丝拧入顶部两个螺丝孔
8. 将平头螺丝拧入远离侧键一侧的两个螺丝孔
9. 将组合螺丝拧入最后两个螺丝孔
10. 将另一侧侧键使用相同方式安装完毕
